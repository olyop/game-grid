# Game Grid
A unoffical NBA scheudle viewer

## Installation
```
git clone https://github.com/olyop/game-grid.git game-grid
cd game-grid
npm i
npm start
```
Go to localhost:3000

## Create Production Build
```
npm run build
node index.js
```
Go to localhost:9000
